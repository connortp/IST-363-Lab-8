Remove Completed Items Forever

  • Attach a click event listener using jQuery onto the buttons for completed todo list items in a similar way as Step 8
  • In the annonymous function, traverse up to the list item again and instead of using the appendTo method, use .fadeOut() to fade the list item element to nothing. The fadeOut method takes two parameters
    • The first parameter is the duration of the fade in milliseconds. Set this to 500
    • The second parameter is optional. It's called a callback function. This function runs once the fade transition has been completed. Inside this callback function, use the .remove() jQuery method to actually remove this list item from the DOM completely

You now have a working simple To-Do list written in JavaScript and jQuery!

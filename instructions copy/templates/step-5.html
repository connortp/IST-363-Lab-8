Click Event Listener

  • When you click the next to the text input, a new to-do list item should be constructed. An event listener will need to be attached to this button, so we can run the necessary code whenever we detect a click on the button
  • Using jQuery, attach a click event handler to the button and pass through an annonymous function with an event parameter. This function will run whenever the user clicks the button
  • In the annonymous function, use event.preventDefault() to prevent the button from submitting the form and having the page reload
  • Create a variable named todoText and assign it the value of the current text of the input. Use jQuery to select the input using a CSS selector and then the val() jQuery method to get the value of the text input
  • Next we need to be able to take this todoText variable value and create an item out of it. Since we also want to be able to quickly add new to-do list items by also pressing enter, instead of putting all the code for creating the item here and then in a second place, let's use a function to store this repeated functionality.

Getting Started

  • Whenever when working with jQuery and JavaScript, it's always a good idea to put all the code into a document ready function. jQuery will hold off on running any code inside a document ready statement until the DOM has been constructed. This allows you to not worry about any loading or timing issues and your script will run as intended no matter where you put it on the page
  • In todo.js create a jQuery document ready function

addTodo Function

  • Define a new function named addTodo and give it a paramter called todoText. This parameter can be named anything, but it's useful to give it a name that matches the variable or type of data we expect to pass to it
  • Some basic validation is needed to prevent creating empty to-do list items (If a user clicks the without any text entered into the input). Create an if statement where the condition is just todoText.
    • When a condition is just a variable, it will return true as long as the variable is defined and has some assigned value that is not 0, false, or nothing. In our case, todoText will be nothing if the input is empty and return false, or if it's a string it'll evaluate to true
  • Inside the if statement, use jQuery to append a new <li> to the "Current To-Dos" List
  • The list item that gets appended should contain a text value of the todoText variable, and also a button that will allow marking it as complete
  • Also, for good UI/UX, clear the value of text input by setting its value to nothing (an empty string)

Mark Items As Completed

  • When a user clicks on the , the to-do item should be removed from the Current list and added to the completed list
  • Attach a click event handler to button on list items in the #todo-list element. Since these todo list items do not all exist when our document ready function fires, the event handler can't be attached directly to the button. Use jQuery's on() method to indirectly create the click event handler and pass it an annonymous function
  • Inside the annonymous function, use jQuery to traverse from the button that was clicked on ($(this)) to get to the parent list item and then move it to the completed list element
